## Segment Tree
- 완전 이진 트리의 특성상, ( h >= 1) , 2^(h-1) < N <= 2^h 이 성립
  모든 항에 log2를 해주면 h-1 < log2(N) <= h 이기 때문에 올림을 해주면 높이 h 값을 구할 수 있습니다.
-  h 인 포화 이진 트리의 전체 크기는 (2 ^ (h+1) ) - 1
